1. Installing GoLang on macOS

Follow these steps to install GoLang on your macOS system:

  1. Visit the official GoLang website: https://golang.org/dl/
  2. Download the macOS installer (e.g., go1.20.5.darwin-amd64.pkg).
  3. Open the downloaded package and follow the installation prompts.
  4. Verify the installation by opening Terminal and typing:
    5. go version

    You should see output similar to:

    go version go1.20.5 darwin/amd64

2. Creating a Hello World Application

Let's create a simple "Hello World" application in Go:

  1. Open Terminal.
  2. Create a new directory for your project:
    3. mkdir hello-world
cd hello-world
  3. Create a new file named main.go:
    4. touch main.go
  4. Open main.go in your preferred text editor and add the following code:
    5. package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
  5. Run the application:
    6. go run main.go

    You should see the output:

    Hello, World!

3. Serving This Website Using npm Web Server

To serve this tutorial website locally using an npm web server, follow these steps:

  1. Ensure you have node.js and npm installed. You can verify by running:
    2. node -v
npm -v
  2. If not installed, download and install Node.js from https://nodejs.org/.
  3. Navigate to the project directory and install dependencies:
    4. npm install
  4. Start the web server:
    5. npm start
  5. Your default web browser should automatically open the website. If not, navigate to http://127.0.0.1:8080.
